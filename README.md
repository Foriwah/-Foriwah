## 🔑 Key Challenges I've Solved

**Low-bandwidth performance** — GTI Mobile Banking needed to be reliable in São Tomé where connectivity is inconsistent. I optimised API call efficiency, implemented intelligent caching, and refactored polling flows to reduce unnecessary network requests.

**Core banking integration** — Connecting mobile frontends to T24 core banking systems requires careful handling of authentication, session management, and real-time transaction state. I've built and maintained these integrations across multiple projects.

**Security hardening** — Fintech apps are high-value targets. I've implemented SSL pinning, emulator detection, root detection, and developer mode detection to protect production banking applications.

**Cross-market UI/UX** — Building for users in Ghana, Nigeria, and São Tomé means different languages, connectivity contexts, and usage patterns. I've designed and tested interfaces that work across these diverse environments.
